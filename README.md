# 📊 Task 2: Feature Engineering & Model Optimization

## 📌 Project Overview
This project focuses on house price prediction using the California Housing dataset. It includes feature engineering, data preprocessing, and model optimization using multiple regression algorithms.

The goal is to compare different models and identify the best-performing model based on evaluation metrics.

---

## 🧠 Models Used
- Linear Regression  
- Ridge Regression  
- Decision Tree Regressor  

---

## ⚙️ Workflow
1. Data Collection (California Housing Dataset)  
2. Data Preprocessing  
3. Feature Engineering  
4. Model Training  
5. Model Evaluation  
6. Performance Comparison  

---

## 📊 Evaluation Metrics
- Mean Squared Error (MSE)  
- Root Mean Squared Error (RMSE)  
- R² Score  

---

## 🏆 Results

| Model | MSE | RMSE | R² Score |
|------|------|------|----------|
| Linear Regression | 0.556 | 0.746 | 0.576 |
| Ridge Regression | 0.556 | 0.746 | 0.576 |
| Decision Tree Regressor | 0.525 | 0.724 | 0.600 |

---

## 🥇 Best Model
Decision Tree Regressor performed best with:
- Lowest RMSE (0.724)  
- Highest R² Score (0.600)
